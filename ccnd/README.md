This directory contains the new version of Zeta along with an experimental evaluation.

Details are given in the paper 

Zavatteri, M., Combi, C., Viganò, L.: Resource Controllability of Business Processes Under Conditional Uncertainty. Journal on Data Semantics (2021) 

which extends 

Zavatteri, M., Combi, C., Viganò, L.: Resource Controllability of Workflows Under Conditional Uncertainty. In: Business Process Management Workshops, pp. 68–80. Springer (2019). DOI 10.1007/978-3-030-37453-2 7

================
We provide binaries for FreeBSD, Linux, and Windows. For dynamically linked binaries libz3 is required.

zeta.bsd: statically-linked freebsd binary
zeta.win: dynamically-linked windows binary
zeta.lnx: dynamically-linked linux binary

'zeta' is a symbolic link to zeta.bsd
